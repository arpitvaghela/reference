Syntax Convention
<"type-of-value">
(/) (,) to be used literally
X|Y any one can appear
X||Y either one or both can appear
X&&Y both has to appear
[..] for grouping

(*) repeated zero or more times
(+) repeated one or more times
(#) repeated one or more times (,)comma seperated
(?) optional (zero or one time)
{m,n} repeated at least m and atmost n times
